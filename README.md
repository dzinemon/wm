# Jekyll Bootstrap 4.5.2

Jekyll + Gulp 4 + Bootstrap 4
and Browser-sync + 

## Steps to start
- (Install Jekyll)[https://jekyllrb.com/docs/installation/]
- NodeJS
- NPM
- Gulp

## Installation
1. Clone or download this respository.
2. Run `npm install`
3. Run `bundle install`
4. Run `gulp` to start developing
